# Changelog

## v1.0.0 - 2026-06-20

First formal GitHub release of GaoGao Prompt.

首个正式 GitHub 发版。

### Added

- Prompt-director workflow for GPT Image / GPT Imagine prompt writing.
- Fast router from user intent to output contract.
- Brief classifier for image type, mode, preservation risk, text risk, and series risk.
- Output contracts for final prompts, visual direction sets, prompt rescue, image edits, multi-reference prompts, structured layouts, series bibles, and generated-image reviews.
- Progressive reference files for prompt recipes, model settings, style vocabulary, craft rules, examples, and templates.
- Commercial guardrails for product truth, exact text, no invented claims, and post-production fallbacks.
- Release package, manifest, validation report, and bilingual public README.
